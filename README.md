# 📊 Sales Dashboard (FY 2024–2025)

An interactive and visually modern **Sales Analytics Dashboard** built using **HTML, CSS, and JavaScript (Chart.js)**.  
This dashboard provides insights into **sales, cost, profit, transactions, and product performance** across India.

---

## 🚀 Live Overview

This project simulates a real-world **Computer Hardware Sales Dashboard** with:

- 📈 KPI metrics (Sales, Cost, Profit, Transactions, Quantity)
- 📊 Category-wise product analysis
- 🏷️ Brand-wise sales distribution
- 🌍 State-wise sales performance
- 👨‍💼 Supervisor-based filtering system

---

## ✨ Key Features

### 🔹 Interactive Dashboard
- Dynamic filtering by **Supervisor**
- Real-time updates of charts & KPIs

### 🔹 Data Visualization
- Bar Charts → Category & State Sales
- Doughnut Chart → Brand Sales Distribution
- Cost Analysis by Brand

### 🔹 KPI Cards
- Total Sales (₹)
- Total Cost (₹)
- Total Profit (₹)
- Transactions
- Quantity Sold

### 🔹 Clean UI/UX
- Dark modern theme 🌙
- Responsive layout (mobile-friendly)
- Smooth hover effects & animations

---

## 🛠️ Tech Stack

- **HTML5** – Structure  
- **CSS3** – Styling (Custom Dark Theme UI)  
- **JavaScript (Vanilla JS)** – Logic & Interactivity  
- **Chart.js (v4)** – Data Visualization
